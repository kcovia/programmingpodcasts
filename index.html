  • 5 by 5 an Internet broadcasting network.

  • The Changelog is a member supported blog, weekly newsletter and podcast that covers the intersection of software development and open source.

  • Debug is a conversational interview show about developing software and services, primarily for iPhone, iPad, Mac, and gaming. Hosted by Guy English and Rene Ritchie, it's all the great talk you get at the bar after the conference, wrapped up in convenient podcast form.

  • Dev1.tv is a podcast focusing on entry level developers.

  • .NET Rocks is an Internet audio talk show for .NET developers.

  • FLOSS Weekly is all about Free Libre Open Source Software. Join host Randal Schwartz and his rotating panel of co-hosts every Wednesday as they talk with the most interesting and important people in the Open Source and Free Software community.

  • Hanselminutes Podcast is "Fresh Air" for developers. Scott Hansel interviews movers and shakers in technology in this commute-time show. From Michio Kaku to Paul Lutus, Ward Cunningham to Kimberly Bryant, Hanselminutes is talk radio guaranteed not to waste your time..

  • Herding Code is a weekly developer discussion featuring K. Scott Allen (odetocode.com), Kevin Dente, Scott Koon (lazycoder.com), and Jon Galloway.

  • How to Program with Java will teach you step by step how to use the Java programming language to create your own applications or web applications! These Java tutorials are presented in plain English and explain all of the important Java programming concepts needed to excel in the field of software.

  • The Java Posse is a podcast of news, discussion, and interviews about the Java programming language and associated Java technologies.

  • JavaScript Jabber is a podcast focusing on the JavaScript language.

  • Mostly Erlang is a podcast about Erlang and sometimes other functional programming.

  • Mostly Lazy is a Clojure podcast... when they get around to it.

  • NodeUp is a Node.js podcast.

  • On Computing is a podcast featuring Grady Booch, software engineer. In this podcast, Brooch reads from his On Computing columns from IEEE Software magazine.

  • PHP Town Hall is a way for PHP developers to raise questions about current events (or upcoming things) in the PHP community, with different guests each week.

  • Programming Throwdown attempts to educate Computer Scientists and Software Engineers on a cavalcade of programming and tech topics. Every show will cover a new programming language, so listeners will be able to speak intelligently about any programming language.

  • Ratchet and the Geek is a podcast about the intersection of social media, pop culture, and technology.

  • Relevance is a podcast from the software company Relevance. In this podcast, Relevance's Craig Andera interviews guests like Rich Hickey, the author of Clojure, Jason Wolfe from Prismatic, Stu Halloway, and Michael Fogus. Listeners will also hear about the business of cloud native software development and the latest advances in the Clojure language.

  • Ruby Rogues - Rubyist.where(:rogue => true).limit(6).all.talk(:about => Topics.where(:awesome => true))

  • Security Now - Steve Gibson, the man who coined the term spyware and created the first anti-spyware program, creator of Spinrite and ShieldsUP, discusses the hot topics in security today with Leo Laporte.

  • Software Engineering Radio is a podcast targeted at the professional software developer. The goal is to be a lasting educational resource, not a newscast. Every two to four weeks, a new episode is published that covers all topics software engineering. Episodes are either tutorials on a specific topic, or an interview with a well-known expert from the software engineering world.

  • The Stack Exchange Podcast gives you an unparalleled look inside the building and running of one of the web's hottest startups: Stack Exchange. Instead of the typical podcast format, Jeff and Joel are joined by a different guest each week as they discuss the strategy and direction of Stack Exchange, the decisions they've made about the community and where things are going next.

  • This Developer's Life is a podcast about developers and their lives.

  • This Week on Channel 9 is a podcast for developers straight from Microsoft.

  • The Treehouse Show is your weekly dose of web design and web development news, hosted by Nick Pettit (@nickrp) and Jason Seifer (@jseifer) of Team Treehouse.

  • Yet Another Podcast is a podcast by Jesse Liberty focusing on intelligent technical conversation.
